## 🚀 Cómo Ejecutar

### Desarrollo Local
```bash
cd C:\Arquitectura\systems-engineer-portfolio
npm install
npm run dev
# Abre http://localhost:3000
```

### Build Producción
```bash
npm run build
# Output: carpeta dist/ (108KB optimizado)
```

### Deploy
**Opción 1: Vercel (Recomendado)**
```bash
npm install -g vercel
vercel
```

**Opción 2: Netlify**
```bash
npm install -g netlify-cli
npm run build
netlify deploy --prod --dir dist
```

**Opción 3: GitHub Pages**
```bash
npm run build
git add dist/
git commit -m "Deploy"
git push
```
